Deleting a VBS Backup (Deprecated)
==================================

Function
--------

This API is used to delete a VBS backup. This API is deprecated. You are advised to use the API described in :ref:`Deleting a VBS Backup (Native OpenStack API) <en-us_topic_0049453221>` to delete a backup.

URI
---

-  URI format

   POST /v2/{project_id}/cloudbackups/{backup_id}

-  Parameter description

   ========== ========= ==============================
   Parameter  Mandatory Description
   ========== ========= ==============================
   project_id Yes       Project ID
   backup_id  Yes       ID of the backup to be deleted
   ========== ========= ==============================

Request
-------

None

-  Parameter description

   | Parameter             | Type                  | Description                                                                     |

   +-----------------------+-----------------------+---------------------------------------------------------------------------------+
   | message               | string                | Error message returned after an error occurs                                    |
   +-----------------------+-----------------------+---------------------------------------------------------------------------------+
   | code                  | string                | Error code returned after an error occurs                                       |
   |                       |                       |                                                                                 |
   |                       |                       | For details about error codes, see :ref:`Error Codes <en-us_topic_0022472083>`. |
   +-----------------------+-----------------------+---------------------------------------------------------------------------------+
